プライバシーポリシー

「ポめも」（以下「本アプリ」）は、ポイントサイトの案件管理を目的とした個人開発アプリです。本プライバシーポリシーでは、本アプリにおける情報の取り扱いについて説明します。

1. 収集する情報

本アプリは、ユーザーの個人情報（氏名、メールアドレス、電話番号等）を収集しません。

ユーザーが本アプリ内で入力したデータ（案件名、ポイント情報、メモ等）は、すべてお使いの端末内にのみ保存され、外部サーバーへ送信されることはありません。

2. 広告について

本アプリでは、広告配信のために AppLovin MAX を利用しています。AppLovin は広告の最適化を目的として、デバイス識別子等の情報を収集する場合があります。

iOS端末では、App Tracking Transparency（ATT）に基づき、トラッキングの許可をお尋ねします。許可しない場合でもアプリの全機能をご利用いただけます。

AppLovinのプライバシーポリシー: https://www.applovin.com/privacy/

3. 利用する外部サービス

  • AppLovin MAX - 広告配信
  • Firebase - アプリの基盤サービス（分析機能は無効化しています）
  • Apple iTunes Search API - アプリ検索機能（検索キーワードのみ送信、個人情報は含みません）

4. データの保存と削除

本アプリのデータはすべてお使いの端末内に保存されます。クラウドへの同期やバックアップは行いません。

アプリを削除（アンインストール）することで、保存されたすべてのデータが完全に削除されます。

5. プライバシーポリシーの変更

本ポリシーは、必要に応じて変更される場合があります。重要な変更がある場合は、アプリ内でお知らせします。
